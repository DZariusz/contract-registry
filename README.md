# Contract Registry

## Overview

`ContractRegistry.sol` solution comes with predefined interfaces 
for other contracts that need to be updatable.

## Install

```
git clone <this-repo>
git hf init
npm install
```

## Testing

```
npm run lint
npm run test
```
